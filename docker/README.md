# Docker:

## Commands
 - docker --help
 - docker images  (shows all the images present)
 - docker pull <image> (get image from registry eg: alpine, ubuntu)
 - docker pull <image>:<tag> (specify a version of an image)
   Example : docker pull ubuntu:16
